# React E-Commerce Project

A fully responsive e-commerce frontend application built with React.

## 🛠️ Technologies Used
- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Bootstrap

## ✨ Features
- Product listing page
- Shopping cart
- Checkout process
- Fully responsive design

## 🚀 Getting Started
```bash
npm install
npm start
```
